About the Challenge

Ready to dive into the world of web development with Django? This challenge is your chance to learn the fundamentals and build a real-world web application in just one day!

Event Structure:

Session 1: Django Fundamentals

  • Introduction to HTML, CSS, JavaScript, and Django
  • Setting Up Your Django Project
  • Django MVT Architecture
  • Creating Models and Admin Interface
  • Handling Front-End Data with AJAX

Session 2: Build & Deploy

  • Teams will be formed and assigned a project
  • Apply your Django skills to build a functional web app
  • Learn to deploy your project on Render (free hosting)

Outcomes:

  • Understand the power and efficiency of Django for web development
  • Gain practical, hands-on experience in building a web application from scratch
  • Learn deployment strategies to make your project live
  • Network with peers and mentors, and get inspired!

Don't miss this opportunity to level-up your web development skills. Register now and get ready to code!
